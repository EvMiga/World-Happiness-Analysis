# World Happiness Analysis

## Project Overview
This project provides a multi-dimensional analysis of the **World Happiness Report**, a landmark survey that measures the state of global happiness. The objective of this study is to explore how various economic, social, and health-related factors influence happiness levels across different countries and regions. 

The analysis is presented through an interactive dashboard that visualizes key correlations and global trends.

**Dataset Source:** [World Happiness Report (Kaggle)](https://www.kaggle.com/datasets/unsdsn/world-happiness) under CC0: Public Domain license.

## Key Visualizations
The project features a comprehensive interactive dashboard built with `Plotly`, including:
* Top 10 Comparison:** A bar chart comparing countries with the highest GDP per capita against their Health (Life Expectancy) scores.
*  Correlation Heatmap:** A visual matrix showing the relationships between GDP, Family, Health, Freedom, Trust in Government, Generosity, and the Happiness Score.
*  Regional Analysis:** A scatter plot demonstrating the dependency of Happiness Scores on GDP per capita, color-coded by global regions.
*  Global Choropleth Map:** A world map displaying the distribution of GDP per capita with interactive tooltips for life expectancy.

## Key Insights
1.  **Economic Impact:** There is a strong positive correlation between GDP per capita and the overall Happiness Score.
2.  **Regional Clusters:** Countries in Western Europe and North America consistently show higher happiness indices compared to other regions.
3.  **Holistic Wellbeing:** While wealth is a major factor, social support (Family) and Health (Life Expectancy) are equally critical drivers of a nation's happiness.
4.  **Actionable Data:** The dashboard helps identify regions where policy improvements in health or social freedom could potentially yield the highest increase in life satisfaction.
